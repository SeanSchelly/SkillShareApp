# SkillShareApp
A skill-share app that matches students and teachers by interest and experience.

# System Design
Our Skillshare system provides a basic console-based interface, allowing the user to pick between a number of actions, ranging from sending a skill request to adding reviews/ratings. Request handling is also implemented, via the use of: java.util.ArrayList; java.util.List packages. These imports form part and parcel of request functionality in the project. Search facility and dashboard are also added. The system uses interfaces (Serializable) to enable object serialization for data persistence. It also attempts to provide functionality for local database connectivity, while also providing DB 'simulators' in code via arrays. A number of classes such as Profile, TutorProfile, Person, Skills, and User make it possible for users to create a profile. Abstraction is also demonstrated via a basic File IO operation. Scanners are used regularly for input, for loops and if statements along with switch cases are used for automation and decision-making logic. 

# How to Run Documentation
1. Download .zip format
2. Extract the .zip
3. Run command prompt and enter directory
4. compile main: **javac Main.java**
5. run main: **java Main.java** (or simply run main)
_TIP: When running the search option for choice 13, the search is case sensitive and the more the detail, the more accurate the search._
