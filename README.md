# SkillShareApp
A skill-share app that matches students and teachers by interest and experience.

# System Design
Our Skillshare system provides a basic console-based interface, allowing the user to pick between a number of actions, ranging from sending a skill request to adding reviews/ratings. Also implements a search functionality that makes use of a search facility and dashboard. The system uses interfaces (Serializable) to enable object serialization for data persistence. It also attempts to provide functionality for local database connectivity, while also providing DB 'simulators' in code via arrays. A number of classes such as Profile, TutorProfile, Person, Skills, and User make it possible for users to create a profile. Abstraction is also demonstrated via a basic File IO operation. Scanners are used regularly for input, for loops and if statements along with switch cases are used for automation and decision-making logic. Request handling is also implemented, via the use of: java.util.ArrayList and java.util.List packages. These imports form part and parcel of request functionality in the project.

# How to Run Documentation
1. Clone Repostiory to a local directory
2. Extract the .zip if downloaded in that format
3. Run command prompt and enter directory
4. compile main: **javac Main.java**
5. run main: **java Main.java**
